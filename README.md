# The ADC Archiver Project

Welcome to the ADC Archiver Project!

## What is the ADC Archiver Project?

ADC Archiver is an archiving tool that has very simple headers and file blocks. Making it future proof and easy to use.

## Why use the ADC Archiver Project?

The ADC Archiver Project is an open source project that is free to use.

## How do I use the ADC Archiver Project?

To use the ADC Archiver Project, you need to have the ADC Archiver installed on your computer.

You can download the ADC Archiver Project from [here](https://github.com/Mealman1551/ArchivedDataCodec/releases/latest)

## How do I contribute to the ADC Archiver Project?

You can contribute to the ADC Archiver Project by forking the [canary repository](gitlab.com/Mealman1551/adc-canary) and making a pull request.

## How do I report a bug?

You can report a bug by opening an issue on the [main repository](https://github.com/Mealman1551/ArchivedDataCodec/issues).
